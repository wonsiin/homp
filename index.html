GIT HUB 저장 장법

cd ../../… : 경로를 지정해준다.
git init : git 파일을 관리할수있게 해준다.
git branch -M main : branch의 main 이라는 저장 경로를 생성.
git remote add 경로명 주소 : git의 주소값으로 연결을 시킨다.
git add . : 동기화 안된 데이터들을 임시 저장시켜준다.
git status : 임시 저장이 되었는지 확인한다.
(임시 저장이 되면 초록색, 안되면 빨간색)
git commit -m "내용" : 저장할때 메모를 한다.
git push -u 경로명 branch명 : 해당 branch로 저장을 시킨다.
홈페이지에서 확인 해주면 끝.

데이터 가져오는 방법
git pull 경로명 branch명 : 경로명의 master 의 data를 가져온다.

데이터 매치 하는법.
git merge branch명 : 해당 branch 에서 branch명으로 데이터 동기화 시킨다.
(이렇게 할 경우 머지할때 시간이 걸린다)
(그래서 git 홈페이지에서 merge를 하고 pull 하는것을 추천)
